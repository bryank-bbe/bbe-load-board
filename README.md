# bbe-load-board (v7.1 cloud-ready)

## Deploy on Streamlit Cloud
1. Put these files in repo root:
   - `app.py`
   - `requirements.txt`
   - `runtime.txt`
   - `.streamlit/streamlit.toml`
   - `sample_loads.csv`
   - `zip_db.csv`
   - `brokers.csv`

2. In Streamlit Cloud, set entrypoint = `app.py`.

3. Push to `main` and hit **Restart** if needed.
